# YNote 資料格式說明文件

本文件說明 YNote 記譜格式的解析方式，供模型溝通時作為前置參考資料。

---

## 1. YNote 序列格式

### 1.1 基本結構

每個音符由 **4 個字符** 組成：
```
[音高代碼 2字符][時值代碼 2字符]
```

**範例**：
```
E408G408A408D508
│ │ │ │ │ │ │ └─ 時值: 08 (八分音符)
│ │ │ │ │ │ └─── 音高: D5 (D5音)
│ │ │ │ │ └───── 時值: 08
│ │ │ │ └─────── 音高: A4
│ │ │ └───────── 時值: 08
│ │ └─────────── 音高: G4
│ └───────────── 時值: 08
└─────────────── 音高: E4
```

### 1.2 音高代碼對照表 (pitch)

| 代碼 | 音高 | MIDI | 說明 |
|------|------|------|------|
| 00 | REST | - | 休止符 |
| A3 | A3 | 57 | |
| B3 | B3 | 59 | |
| C4 | C4 | 60 | 中央 C |
| ... | ... | ... | |

### 1.3 時值代碼對照表 (duration)

| 代碼 | 時值 (ticks) | 說明 |
|------|--------------|------|
| 16 | 120 | 十六分音符 |
| 08 | 240 | 八分音符 |
| 04 | 480 | 四分音符 |
| ... | ... | ... | |
